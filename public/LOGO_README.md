# TripNoute Logo

Bu klasöre `tripnoute-logo.png` dosyasını eklemen gerekiyor.

## Logo Özellikleri:
- **Dosya adı:** tripnoute-logo.png
- **Boyut:** 40x40 px (veya daha büyük, otomatik resize edilecek)
- **Format:** PNG (şeffaf arka plan önerilir)
- **Konum:** /public/tripnoute-logo.png

## Kullanım:
Logo şu sayfalarda kullanılıyor:
- Dashboard (/dashboard)
- Map View (/map)
- Add Place (/places/add)
- Edit Place (/places/edit/[id])

## Manuel Kurulum:
1. Logo dosyasını bu klasöre kopyala
2. Dosya adını `tripnoute-logo.png` olarak ayarla
3. Dev server otomatik olarak yenilenecek
